### Mantošana

Šādi izskatās mantošana `Java` valodā:

```java
class Dog extends Animal {
 // some code
}
```

Klase `Dog` manto klasi `Animal`. Atslēgvārds `extends` norāda uz mantošanu.

`Dog` - apakšklase.
`Animal` - virsklase.
