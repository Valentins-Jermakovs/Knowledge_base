### Enums

Tas ir speciāls datu tips, kas definē konstantu kolekciju:

```java
enum Rank {
  SOLDIER,
  SERGEANT,
  CAPTAIN
}
```

Lai piekļūtu pie `enum` datiem, izmanto punkta notāciju:

```java
Rank a = Rank.SOLDIER;
```
