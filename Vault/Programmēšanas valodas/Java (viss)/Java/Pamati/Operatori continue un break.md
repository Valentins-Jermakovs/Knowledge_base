### Operatori continue un break

Operators `break` ļauj lauzt ciklu:

```java
for (int i = 0; i < 10; i++){
    if (i == 5)
        break;
    System.out.println(i);
}
```

Operators `continue` ļauj uzsākt jauno iterāciju:

```java
for (int i = 0; i < 10; i++){
    if (i == 5)
        continue;
    System.out.println(i);
}
```
