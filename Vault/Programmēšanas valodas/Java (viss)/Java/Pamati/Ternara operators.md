### Ternara operators

Sintakse līdzīga `C#` valodai:

```java
// [nosacījums] ? [ja true] : [ja false]
int x=3;
int y=2;
int z = x<y? (x+y) : (x-y);
System.out.println(z); // 1
```

Kur:

- `x<y?` - nosacījums;
- `(x+y)` - izpildās, ja `true`;
- `(x-y)` - izplidās, ja `false.`
