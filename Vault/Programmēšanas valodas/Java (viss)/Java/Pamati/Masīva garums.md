### Masīva garums

Masīvu garuma noteikšanai tiek izmantota īpašība `length`:

```java
int[] nums = {1, 2, 3, 4, 5};
int length = nums.length;   // 5
```

Un, lai iegūt pēdējo masīva elementu:

```java
int last = nums[nums.length-1];
```
