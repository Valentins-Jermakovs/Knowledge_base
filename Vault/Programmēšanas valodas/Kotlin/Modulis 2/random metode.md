# random metode

Šī metode tiek izmantota nejaušo skaitļu ģenerācijai:

```kotlin
randomTap = (2..4).random()
```

Pastāv divi parametri, diapazona sākuma skaitlis un beigu skaitlis. Tie tiek atdalīti ar `..` 2 punktiem.
