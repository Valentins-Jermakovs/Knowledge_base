# `null` vērtība

Lai iestatīt kādam mainīgam `null` vērtību, tā definēšanas laikā izmanto `?` simbolu:

```kotlin
fun main() {
    var number: Int? = 10
    println(number)
    
    number = null
    println(number)
}
```
