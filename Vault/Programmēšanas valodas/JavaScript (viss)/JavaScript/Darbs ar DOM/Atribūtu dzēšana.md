### ⬢ Detalizēts apraksts
#### Atribūtu dzēšana

Lai noņemtu atribūtu, izmantojiet metodi `removeAttribute()`, kurai tiek nodots noņemamais atribūts:

```html
<a id="home" href="https://metanit.com" style="color:navy;">Home</a>
<script>
// получаем элемент
const element = document.getElementById("home");
// удаляем атрибут style
element.removeAttribute("style");
</script>
```
