### ⬢ Detalizēts apraksts
#### `break` un `continue`

##### Operators `break`

Šis operators tiek izmantots, lai pārtraukt cikla darbību:

```js
for(let i = 1; i <= 6; i++){
      
     if(i === 4) break;
     console.log(i);
}
console.log("Конец работы");
```

##### Operators `continue`

Šis operators tiek izmantots, lai uzsākt jauno iterāciju:

```js
for(let i = 1; i <= 6; i++){
      
    if(i === 4) continue;
    console.log(i);
}
console.log("Конец работы");
```
