### ⬢ Detalizēts apraksts
#### Ievads

Lai `html` dokumentā izmantot `JavaScript`, izmanto tegu `<script>`.

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title>METANIT.COM</title>
</head>
<body>
    <script>
        document.write("<h2>Первая программа на JavaScript</h2>");
    </script>
</body>
</html>
```

Komentāri:

- `//` - iekomentē vienu koda rindu;
- `/**/` - ļauj iekomentēt vairākas koda rindas.
