### ⬢ Detalizēts apraksts
#### Cikls `for...of`

Analoģisks `python` valodas `for` ciklam, iterē pa vērtībām.

```js
const text = "Hello";
for(char of text){
      
     console.log(char);
}
```

```js
const people = ["Tom", "Sam", "Bob"];
for(const person of people) {
    console.log(person);
}
```
