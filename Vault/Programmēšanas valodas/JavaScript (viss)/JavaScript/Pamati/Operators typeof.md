### ⬢ Detalizēts apraksts
#### Operators

Šis operators ļauj noteikt mainīga datu tipu.

```js
let id;
console.log(typeof id);   // undefined
 
id = 45;
console.log(typeof id);   // number
 
id = 45n;
console.log(typeof id);   // bigint
 
id = "45";
console.log(typeof id);   // string
```
