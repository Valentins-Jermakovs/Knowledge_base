### ⬢ Detalizēts apraksts
#### `switch...case` konstrukcija

Tas pats, kas `C#` valodā.

```js
const income = 700;
switch(income){
 
    case 100 : 
        console.log("Доход равен 100");
        break;
    case 200 : 
        console.log("Доход равен 200");
        break;
    case 500 : 
        console.log("Доход равен 500");
        break;
    default: 
        console.log("Доход неизвестной величины");
        break;
}
```
