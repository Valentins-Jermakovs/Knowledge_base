#### Ternara pieraksts

```
[nosacījums] ? [vērtība, ja True] : [vērtība, ja False]
```

```js
const a = 1;
const b = 2;
const result = a < b ? a: b;
console.log(result); // 1
```

```js
const a = 1;
const b = 2;
const result = a < b ? a + b : a - b;
console.log(result); // 3
```
