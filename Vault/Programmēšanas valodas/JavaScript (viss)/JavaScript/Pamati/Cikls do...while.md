### ⬢ Detalizēts apraksts
#### Cikls `do...while`

Šis cikls nodrošina, kā bloks `do` izpildās vismaz vienu reizi:

```js
let i = 1;
do{
    console.log(i);
    i++;
}while(i <= 5)
```

Sākumā izpildās bloks `do`, pēc tam notiek nosacījuma pārbaude.
