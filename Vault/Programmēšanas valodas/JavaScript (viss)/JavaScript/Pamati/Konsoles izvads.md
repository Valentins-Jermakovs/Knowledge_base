### ⬢ Detalizēts apraksts
#### Konsoles izvads

Lai izvadīt ziņojums konsolē, izmanto `console.log`

```js
const sum = 5 + 8;
console.log("Результат операции");
console.log(sum);
```

Mainīgo izmantošana konsoles izvadā:

```js
let myVar = 89;
if(myVar){
     
    console.log(`Переменная myVar имеет значение: ${myVar}`);
}
```
