### ⬢ Detalizēts apraksts
#### Funkcijas parametri

Funkcija var pieņemt vairākus parametrus.

```js
function sum(a, b){
    const result = a + b;
    console.log(result);
}
 
sum(2, 6);          // 8
sum(4, 5);          // 9
sum(109, 11);       // 120
```
