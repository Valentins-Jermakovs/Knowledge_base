### ⬢ Detalizēts apraksts
#### Metode `repeat()`

Šī metode ļauj izveidot teksta virkni, veicot kādas teksta virknes atkārtošanu.

```js
const message = "hello ";
console.log(message.repeat(3)); // hello hello hello
```
