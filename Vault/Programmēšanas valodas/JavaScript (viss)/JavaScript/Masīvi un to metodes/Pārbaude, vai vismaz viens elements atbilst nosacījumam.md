### ⬢ Detalizēts apraksts
#### Metode `some()`

Šī metode ļauj uzzināt, vai masīvā eksistē tāds elements, kas atbilst metodes nosacījumam:

```js
const numbers = [ 1, -12, 8, -4, 25, 42 ];
const passed = numbers.some(n => n > 0);
console.log(passed); // true
```
