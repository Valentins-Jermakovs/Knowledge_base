### ⬢ Detalizēts apraksts
#### Metode `filter()`

Šī metode atgriež jauno masīvu no elementiem, kas atbilst nosacījumam:

```js
const numbers = [ 1, -12, 8, -4, 25, 42 ];
const filteredNumbers = numbers.filter(n => n > 0);
console.log(filteredNumbers); // [1, 8, 25, 42]
```
