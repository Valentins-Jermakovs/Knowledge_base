### ⬢ Detalizēts apraksts
#### Masīvu apvienošana

Izmantojot `spread` operatoru, var apvienot masīvus.

```js
const men = ["Tom", "Sam", "Bob"];
const women = ["Kate", "Alice", "Mary"];
const people = [...men, "Alex", ...women];
 
console.log(people);     //  ["Tom", "Sam", "Bob", "Alex", "Kate", "Alice", "Mary"]
```
