### ⬢ Detalizēts apraksts
#### Metode `concat()`

Šī metode tiek izmantota, lai apvienot masīvus. Tā atgriež apvienoto masīvu.

```js
const men = ["Tom", "Sam", "Bob"];
const women = ["Alice", "Kate"];
const people = men.concat(women);
console.log(people);         // ["Tom", "Sam", "Bob", "Alice", "Kate"]
```
