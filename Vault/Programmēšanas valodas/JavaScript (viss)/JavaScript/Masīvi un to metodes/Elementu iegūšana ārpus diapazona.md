### ⬢ Detalizēts apraksts
#### Metode `toSplice()`

Šī metode atgriež masīva elementus, kas atrodas ārpus diapazona.

```js
const people = ["Tom", "Sam", "Bill", "Alice", "Kate"];
const range = people.toSpliced(1, 3);
console.log(range);       // ["Tom", "Kate"]
console.log(people);         // ["Tom", "Sam", "Bill", "Alice", "Kate"]
```
