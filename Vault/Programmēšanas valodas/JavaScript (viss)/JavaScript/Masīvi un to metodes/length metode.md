#### Metode `length`

Šī metode ļauj uzzināt masīva garumu.

```js
const fruit = [];
fruit[0] = "яблоки";
fruit[1] = "груши";
fruit[2] = "сливы";
 
console.log("В массиве fruit ", fruit.length, " элемента");
for(let i=0; i < fruit.length; i++)
    console.log(fruit[i]);
```
