### ⬢ Detalizēts apraksts
#### Par `z-index`

`z-index` ļauj kontrolēt to, uz kāda slāņa atrodas elements. Tādējādi var, piem., novietot attēlu aiz teksta.

```css
img {
  position: absolute;
  left: 0px;
  top: 0px;
  z-index: -1;
}
```
