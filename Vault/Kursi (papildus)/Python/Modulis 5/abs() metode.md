### ⬢ Detalizēts apraksts
#### Metode

Metode `abs()` atgriež absolūto, pozitīvo skaitli.

```python
print(abs(-5))   # Вывод: 5
print(abs(5))    # Вывод: 5
print(abs(-3.14)) # Вывод: 3.14
print(abs(3.14))  # Вывод: 3.14
print(abs(3 + 4j)) # Вывод: 5.0
```
