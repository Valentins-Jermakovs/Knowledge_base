### ⬢ Detalizēts apraksts
#### `break` operators

Operators `break` domāts, lai pārtraukt cikla darbību.

```python
# apstādina ciklu uz 2
for i in range(5):
	if i == 2:
		break
```

#### `continue` operators

Operators `continue` tiek izmantos, lai uzsākt jauno iterāciju, izlaižot kodu, kas tiktu izpildīts.

```python
# izlaiž visus pāra skaitļus
for i in range(10):
	if i % 2 == 0:
		continue
```
