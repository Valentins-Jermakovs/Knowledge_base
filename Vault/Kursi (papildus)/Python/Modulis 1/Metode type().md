### ⬢ Detalizēts apraksts
#### Metode `type()`

Metode `type()` ļauj uzzināt mainīga datu tipu.

```python
text = 'Hello World'
print(type(text))  # <class 'str'>

shift = 3
print(type(shift)) # <class 'int'>
```
