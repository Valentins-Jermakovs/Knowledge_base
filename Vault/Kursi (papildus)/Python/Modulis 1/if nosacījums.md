#### `if` nosacījums

```python
if x != 0:
    print(x)
```

Ja funkcija atgriež `True` vai `False`, to var izmantot pa taisno `if` nosacījumā:

```python
if verify(card):
	print('VALID!')
else:
	print('INVALID!')
```
