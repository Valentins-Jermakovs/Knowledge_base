### ⬢ Detalizēts apraksts
#### Metode

`.isupper()` metode pārbaude, vai simbols ir rakstīts `uppercase` formātā. Ja jā, tad `True`, pretējā gadījumā atgriež `False`.

```python
if char.isupper():
	print('Uppercase!')
else
	print('Lowcase!')
```
