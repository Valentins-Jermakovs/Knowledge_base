#### Metodes ar parametriem

Metodēm var nodot parametrus (mainīgos) un veikt ar tiem dažādas operācijas.

```csharp
CountTo(5);

void CountTo(int max) 
{
	for (int i = 0; i < max; i++)
	{
		Console.Write($"{i}, ");
	}
}
```
