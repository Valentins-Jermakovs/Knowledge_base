### ⬢ Detalizēts apraksts
#### Faila pieslēgšana pie `index.html`

Lai pieslēgt savu `style.css` failu pie `html` dokumenta, rīkojies šādi:

```html
<link rel='stylesheet' href='styles.css'>
```

`href` - ir ceļš līdz tavam failam.
