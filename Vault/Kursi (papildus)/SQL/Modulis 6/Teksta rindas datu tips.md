#### CHAR un VARCHAR

| Tips         | Apraksts                                                  | Simbolu diapazons |
| ------------ | --------------------------------------------------------- | ----------------- |
| `CHAR(X)`    | Satur teksta rindas, garums var tikt fiksēts, norādot to. | No 0 līdz 255     |
| `VARCHAR(X)` | Satur teksta rindas, garums dinamisks                     | No 0 līdz 65,535  |
